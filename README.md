## Make clone to project
```
git clone https://EmadMohDev/cruds.git
```

## Go inside the project
```
cd cruds
```


## Create key
```
php artisan key:generate
```

## Install composer
```
composer install 
```

## Generat data
```
php artisan migrate --seed
```

## Run project
```
php artisan serve
```
